Faoin eagraíocht · About
Ní múineann Athbheochan Labs an Ghaeilge. Déanann sé í a chur ar fáil — i bhfeidhmchláir, i huirlisí, agus i gcomhthéacsanna nach mbeifeá ag súil léi.

There is no shortage of Irish language learning tools. What is missing is infrastructure — dictionary apps, everyday utilities, commercial tools — that treat Irish as a default rather than a subject.

Athbheochan Labs builds open-source software to close that gap. Everything we make is free, fully open, and built to last.

Fealsúnacht · Philosophy
01
Normalú, ní múineadh
We don't build language learning tools. We build infrastructure that makes Irish feel like a natural part of digital life — for speakers who already have the language and just need the tools.
02
Oscailte i gcónaí
Everything we build is open-source, forever. No paywalled features, no proprietary core. The Irish language community should own the tools built for it.
03
Bearna a líonadh
We look for genuine gaps — things that don't exist yet and are genuinely needed — rather than rebuilding what already works well elsewhere.
04
Simplíocht
Simple architecture, minimal dependencies, low barrier to contribution. Anyone with Irish language knowledge or basic development skills should be able to get involved.
Rannchuidiú · Contribute

Ní gá Svelte a bheith agat.

You don't need to be a developer to contribute to Athbheochan Labs. Irish language knowledge is just as valuable as code. Suggest words, fix translations, improve copy, or report something that feels wrong.

Féach ar na tionscadail ar GitHub
Teanga · Language
Suggest words for the word-of-the-day list, flag translation errors, or contribute to the Cnuasach phrase bank. No code required.
Dearadh · Design
Improve layouts, suggest UI copy, or contribute to the shared CSS token system. Plain CSS — no framework knowledge needed.
Cód · Code
SvelteKit, TypeScript, and plain HTML. Issues are labelled good first issue for anyone getting started.
